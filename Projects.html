ROCK, PAPER, SCISSORS GAME USING COMPUTER VISION AND MACHINE LEARNING

A rock, paper, scissors game in which the user plays against the CPU by giving hand gestures as inputs, which are recognized by the system through computer vision and trained machine learning models, which are evaluated against the CPU’s own move (rock, paper or scissors). The result is then displayed to the user in the form of a message and total score of both the CPU and the player.

Technologies used: Python, OpenCV, Machine Learning, PyQt

TEXT SENTIMENT ANALYSER

Technologies used: Python

IMPLEMENTING DIJKSTRA'S ALGORITHM TO FIND OPTIMAL SOLUTION OF A MAZE
